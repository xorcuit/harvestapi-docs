## How to get Profile ID

<CodeGroup>

```javascript JavaScript
const params = new URLSearchParams({
  url: 'https://www.linkedin.com/in/williamhgates',
  apiKey: '<api-key>',
});

fetch(`https://api.harvest-api.com/linkedin/profile?${params.toString()}`)
  .then((res) => res.json())
  .then((data) => {
    console.log(`Bill Gates profile ID: ${data.id}`);
  });
```

```bash cURL
curl --request GET \
  --url 'https://api.harvest-api.com/linkedin/profile?url=https://www.linkedin.com/in/williamhgates' \
  --header 'X-API-Key: <api-key>'
```

```python Python
import requests

url = "https://api.harvest-api.com/linkedin/profile?url=https://www.linkedin.com/in/williamhgates"

headers = {"X-API-Key": "<api-key>"}

response = requests.request("GET", url, headers=headers)

print("Bill Gates profile ID: " + response.json()["id"])
```
