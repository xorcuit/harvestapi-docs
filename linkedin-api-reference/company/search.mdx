## Search by LinkedIn GeoID

<GeoIdSnippet />

```javascript JavaScript
const apiKey = '<api-key>';

fetch(`https://harvestapi.net/api/linkedin/geo-id-search?search=Germany`, {
  headers: { 'X-API-Key': apiKey },
})
  .then((response) => response.json())
  .then((data) => {
    console.log('All matches:', data.elements);
    console.log('Closest match GeoID:', data.id);

    const params = new URLSearchParams({
      search: 'Google',
      geoId: data.id,
      page: '1',
      apiKey,
    });
    fetch(`https://harvestapi.net/api/linkedin/company-search?${params.toString()}`)
      .then((response) => response.json())
      .then((data) => console.log(data));
  });
```
