## Search "Software" posts

<CodeGroup>

```javascript JavaScript
const params = new URLSearchParams({
  search: 'Software',
  postedLimit: '24h',
  sortBy: 'relevance',
  page: '1',
});
fetch(`https://harvestapi.net/api/linkedin/post-search?${params.toString()}`, {
  headers: { 'X-API-Key': '<api-key>' },
})
  .then((response) => response.json())
  .then((data) => console.log(data));
```

```bash cURL
curl --request GET \
  --url https://harvestapi.net/api/linkedin/post-search?search=Software&postedLimit=24h&sortBy=relevance&page=1 \
  --header 'X-API-Key: <api-key>'
```

```python Python
import requests

url = "https://harvestapi.net/api/linkedin/post-search?search=Software&postedLimit=24h&sortBy=relevance&page=1"

headers = {"X-API-Key": "<api-key>"}

response = requests.request("GET", url, headers=headers)

print(response.text)
```
