## Get posts from a LinkedIn profile

<CodeGroup>

```javascript JavaScript
const params = new URLSearchParams({
  profile: 'https://www.linkedin.com/in/williamhgates',
  page: '1',
});
fetch(`https://api.harvest-api.com/linkedin/profile-posts?${params.toString()}`, {
  headers: { 'X-API-Key': '<api-key>' },
})
  .then((response) => response.json())
  .then((data) => console.log(data));
```

```bash cURL
curl --request GET \
  --url https://api.harvest-api.com/linkedin/profile-posts?profilePublicIdentifier=williamhgates&page=1 \
  --header 'X-API-Key: <api-key>'
```

```python Python
import requests

url = "https://api.harvest-api.com/linkedin/profile-posts?profilePublicIdentifier=williamhgates&page=1"

headers = {"X-API-Key": "<api-key>"}

response = requests.request("GET", url, headers=headers)

print(response.text)
```
