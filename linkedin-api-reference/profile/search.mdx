## How to get Company ID

```javascript JavaScript
const params = new URLSearchParams({
  search: 'Google',
  location: 'Australia',
  apiKey: '<api-key>',
});

fetch(`https://harvestapi.net/api/linkedin/company-search?${params.toString()}`)
  .then((res) => res.json())
  .then((data) => {
    const companies = data.elements;
    const firstMatchId = companies[0].id;
    console.log(`LinkedIn ID for Google: ${firstMatchId}`);
  });
```

## Search by LinkedIn GeoID

Sometimes search by location may not give you what you expect, as LinkedIn have some other location for your text query.
For example, `NY` returns `New Zealand` as first match instead of `New York`
(it's based on LinkedIn autocomplete feature, you can try it in UI).

To use the correct location, you can find LinkedIn GeoID first.
GeoID search is free, it will not count against your quota.

```javascript JavaScript
const apiKey = '<api-key>';

fetch(`https://harvestapi.net/api/linkedin/geo-id-search?search=New York`, {
  headers: { 'X-API-Key': apiKey },
})
  .then((response) => response.json())
  .then((data) => {
    console.log('All matches:', data.elements);
    console.log('Closest match GeoID:', data.id);

    // Search for profiles at Google in New York
    const params = new URLSearchParams({
      companyId: '1441',
      geoId: data.id,
      page: '1',
      apiKey,
    });
    fetch(`https://harvestapi.net/api/linkedin/profile-search?${params.toString()}`)
      .then((response) => response.json())
      .then((data) => console.log(data));
  });
```
